RO.ME "3 DREAMS OF BLACK"

RO.ME is a creative interactive WebGL project by Chris Milk and Google that showcased the potential of web technologies for storytelling and interactive content. At that time, WebGL was relatively new and not widely used for interactive web experiences. This project demonstrated the potential of WebGL, which allowed for high-quality 3D graphics and animations directly within a web browser .

Technology: HTML 5, three.js and WebGL.
