RO.ME "3 DREAMS OF BLACK"

RO.ME is an interactive film that runs directly in our web browser. The demo was created when the visual models were put together into a three dream sequence allowing the user to experience an impactful interaction while immersing into the scene. The real-time rendering of the models and the changes to the viewing angle achieved through the JavaScript code is quite fascinating.

Technology: 2D Graphics, 3D Graphics, Webgl, JavaScript, HTML5, three.js, GLTF2Loader.js !
