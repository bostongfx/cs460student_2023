RO.ME "3 DREAMS OF BLACK"

RO.ME is a JavScript project that ultilizes WebGL to render videos, 2D, and 3D animations to create a music video for the song 3 Dreams Of Black. Before watching the demo, I could never imagine that web applications could be this powerful, ie. rendering 2D and 3D models in real-time while letting the user interacting with the render at the same time. I'd always thought of web application as a static format with maybe a few interative 3D elements here and there. But actually has been this powerful since 2011 when WebGL came out, and now with WebGPU being the rising successor, this technology would only get more and more powerful from here. This technology could open up so many new possibilities for online media both for the users and for the artists or creators. But I'm just wondering why this technology are not more widely implemented today? I'm more of a Back-end guy but I do have some HTML and JavaScript knowledge but I haven't really heard of WebGL before this class. I guess the closest example of something using interactable web based application that I can think of is Google Maps Street View.

Technology: WebGL, JavaScript, Three.js, HTML5, Hardware Acceleration.
